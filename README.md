# Quiz App

A modern, animated, and responsive **Quiz Application** built using **React JS**, featuring:
- useState
- conditional rendering
- score calculation
- beautiful UI
- animations using Framer Motion

 ✔ 10 Multiple Choice Questions  
 ✔ Score Tracking

# How to Install & Run

1️.Step — Create project folder**
mkdir quiz-app
cd quiz-app

2️.Step — Create Vite project
npm create vite@latest

3.Step — Install dependencies
npm install
npm install framer-motion

4.Step — Run project
npm run dev
